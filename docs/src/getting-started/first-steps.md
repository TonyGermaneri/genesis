# First Steps

Learn the basics of playing Genesis.

## Starting a New Game

1. Launch Genesis
2. Select **New Game** from the main menu
3. Choose a world seed (or leave blank for random)
4. Select difficulty
5. Click **Create World**

### Status Bars
- **Health**: Your life. Reaches zero = death
- **Energy**: Depletes with actions, regenerates over time
- **Hunger**: Eat food to maintain

## Basic Controls

| Key | Action |
|-----|--------|
| W/A/S/D | Move forward/left/back/right |
| Mouse | Look around |
| Left Click | Primary action (attack/use tool) |
| Right Click | Secondary action (place/interact) |
| E | Interact with objects |
| I | Open inventory |
| C | Open crafting menu |
| M | Open map |
| Tab | Show player stats |
| Esc | Pause menu |

## Your First Day

### 1. Gather Resources

Look for trees and rocks nearby:
- **Punch trees** to gather wood
- **Pick up stones** from the ground
- **Collect plant fibers** from grass

### 2. Craft Basic Tools

Open the crafting menu (C):

```
Wood Plank (2)     <- 1 Wood Log
Stone Axe (1)      <- 2 Stone + 3 Wood Plank + 2 Plant Fiber
Stone Pickaxe (1)  <- 3 Stone + 2 Wood Plank + 2 Plant Fiber
```

### 3. Build Shelter

Before nightfall:
1. Gather 20+ wood planks
2. Find a flat area
3. Open build menu (B)
4. Place walls and a door
5. Add a roof

### 4. Survive the Night

At night, hostile creatures spawn:
- Stay inside your shelter
- Craft a torch for light
- Rest until dawn

## Key Mechanics

### Crafting

The crafting system uses:
- **Recipes**: Predefined combinations of items
- **Stations**: Some items require crafting stations (forge, workbench)
- **Quality**: Better tools/materials produce higher quality items

### Inventory

- 40 slots for general items
- Items stack up to 99 (varies by type)
- Hotbar (1-0 keys) for quick access

### Building

- **Snap-to-grid**: Buildings align automatically
- **Structural integrity**: Support structures properly
- **Upgrades**: Improve buildings over time

### Economy

- Trade with NPCs at settlements
- Prices fluctuate based on supply/demand
- Build your own shop to sell items

## Tips for Beginners

1. **Don't venture far on day one** - Explore after you have shelter
2. **Always carry food** - Hunger drains health when empty
3. **Light is important** - Enemies avoid lit areas
4. **Talk to NPCs** - They offer quests and valuable information
5. **Save frequently** - Use the pause menu to save

## Next Steps

- Explore the world and find settlements
- Join a faction for unique benefits
- Set up automated resource gathering
- Discover rare materials in caves
